# lilQR

Free QR code generator (URL, vCard, WiFi, email, SMS, and more) with UTM tracking and PNG/SVG export. A free tool by [lilAgents](https://lilagents.com).

**Live version:** https://lilagents.com/tools/qr/

This is the standalone, open-source build, a self-contained Astro site. The canonical hosted version is folded natively into the lilAgents website.

## Run locally

```
pnpm install
pnpm dev      # dev server
pnpm build    # production build -> dist/
```

## Tech

Astro + Tailwind CSS + vanilla JS. 100% client-side; no backend, no API keys.
